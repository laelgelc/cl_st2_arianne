1. United Nations
Página de índice inicial: https://unfccc.int/decisions?f%5B0%5D=body%3A1343
618 resultados

A página de índice é dinâmica cuja extração de dados é possível através do pacote 'Selenium'. Cada resultado (Decisão) da página de índice provê um link para o relatório (documento PDF), às vezes em várias opções de idioma, que contém a decisão e outro, para uma página que fornece detalhes sobre o relatório.

Da página de detalhes, os seguintes metadados podem ser extraídos:

- Source: UNFCCC
- Publication (data)
- Conference (título da conferência)
- Year of the conference (derivado do metadado 'Conference')
- Session (COP n)
- English PDF link
- Text (derivado de 'English PDF link')


2. Greenpeace, WWF e WRI
2.1. Greenpeace
Página de índice inicial: https://www.greenpeace.org/international/story/page/1/
Página de índice final: https://www.greenpeace.org/international/story/page/130/
130 páginas de índice

Cada página de índice contempla cerca de 10 índices para documentos 'Stories'. Cada um destes índices fornece um link (URL) que leva ao respectivo documento e uma série de hashtags.

Cada documento é uma página web estática de onde o texto, bem como os seguintes metadados podem ser extraídos:

- Source: Greenpeace
- Label
- Title
- Date
- Text
- Hashtags

2.2. WWF
Página de índice inicial: https://www.worldwildlife.org/stories?page=1&threat_id=effects-of-climate-change
Página de índice final: https://www.worldwildlife.org/stories?page=5&threat_id=effects-of-climate-change
5 páginas de índice

O site WWF rejeita a extração de dados programática (403 Client Error: Forbidden). Para contornar este problema, a extração de dados dos documentos deverá ser realizada através do pacote 'Selenium', que simula operações manuais de web browser.

Detalhes importantes:

- Como o 'Selenium' controlará o browser Microsoft Edge (em um computador Windows) é necessário utilizar a distribuição Python (Anaconda) para Windows;
- É necessário envolver o caracter "&" entre aspas duplas: https://www.worldwildlife.org/stories?page=1"&"threat_id=effects-of-climate-change.

Cada página de índice contempla cerca de 25 índices para documentos 'Story'. Cada um destes índices fornece um link (URL) que leva ao respectivo documento e alguns fornecem a data de publicação. Sugerimos capturar dados somente dos documentos que apresentam data.

Cada documento é uma página web estática de onde o texto, bem como os seguintes metadados podem ser extraídos:

- Source: WWF
- Title
- Date
- Text

2.3. World Resources Institute
Página de índice inicial: https://www.wri.org/resources/topic/climate-53/type/insights-50?page=0
Página de índice final: https://www.wri.org/resources/topic/climate-53/type/insights-50?page=64
65 páginas de índice
1293 Recursos

Cada página de índice contempla cerca de 20 índices para documentos 'Insight'. Cada um destes índices fornece um link (URL) que leva ao respectivo documento e uma série de palavras-chave. Note que as páginas de índice selecionam 'Insights' cujas palavras-chave contenham a palavra 'Climate' mesmo que o tópico do 'Insight' não seja 'Climate'.

Cada documento é uma página web estática de onde o texto, bem como os seguintes metadados podem ser extraídos:

- Source: WRI
- Title
- Date
- Topic
- Region (if present)
- Text


Cada website tem suas peculiaridades, o que nos obriga a empregar estratégias diferentes de extração de dados. Sugerimos considerar o trabalho em cada website como um subprojeto a fim de evitar concentração de esforço e tempo em uma única empreitada.
